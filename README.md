# Auto-Absen Extension

By isan

## Deskripsi
Ekstensi browser kustom untuk mempermudah dan mengotomatisasi proses absensi pada portal e-learning.

## Cara Instalasi
1. Ekstrak folder ekstensi ini.
2. Buka browser (Chrome/Edge/Brave) dan masuk ke halaman manajemen ekstensi (`chrome://extensions` atau `edge://extensions`).
3. Aktifkan fitur **Developer mode** di pojok kanan atas.
4. Klik tombol **Load unpacked**.
5. Pilih folder tempat kamu mengekstrak file ini.
6. Ekstensi siap digunakan.
